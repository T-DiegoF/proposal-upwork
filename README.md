# CrewCore MVP

Proposal-grade demo for a construction crew management platform, built as a freelance pitch on Upwork.

**Live:** https://proposal-upwork.vercel.app

---

## What it is

CrewCore is a concept for a mobile-first operations platform aimed at construction companies. The goal was to show a prospective client what a real product could look like — not a generic mockup, but something that actually sells a direction.

The UI is designed around real field conditions: large actions, minimal navigation, role-aware flows. Workers check in fast. Managers see what's happening same-day. Clients get visibility without needing to call anyone.

## Features demonstrated

- **Crew Check-In** — Geolocation-verified attendance with shift summary and phase context
- **Operational Control** — Task assignment organized by project and construction phase
- **Evidence & Reporting** — Daily reports, photo/video capture, and internal communication tied to the right project thread
- **Role-based access** — Four user types: Workers, Managers, Admins, Clients

## Stack

- Next.js 16 / React 19 / TypeScript
- Deployed on Vercel

## Structure

```
app/              → Main Next.js app (landing + feature pages)
crewcore-mvp/     → Static MVP screens (standalone HTML prototype)
```

## Notes

This is a proposal/demo — not a production app. The intent was to demonstrate product thinking, UI decisions, and frontend execution quality to win a freelance contract.
